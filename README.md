# Odin Recipes

This is my first website project from [The Odin Project](https://www.theodinproject.com/).

I built a simple recipe website with a homepage and three recipe pages.

## Live Preview
👉 [View the website here](https://pri-cd.github.io/odin-recipes/)

## What I Learned
- How to structure a website with HTML
- Linking multiple pages together
- Adding and styling images
- Using lists and basic CSS

## Recipes Included

- Coconut Oil Sugar Cookies
- Kheer (Rice Pudding)

## Technologies Used

- CSS3
